使用github搭建个人博客

使用Github搭建个人博客

1.注册一个Github账号，并创建新的库，创建成功会出现下图：

2.要安装搭建环境
下载node以及Git的安装包并进行安装，安装完毕后可使用命令node -v ˎnpm-v和命令git -version分别检验node和Git是否安装成功。
如果安装成功，则如下图：

3.设置ssh
（1）创建一个文件夹，我在D盘下anzhuang文件夹中新建了文件夹blog,然后点击进入Git Bash 界面。

（2）全局配置设置更新源。
（3）设置全局配置user.name和user.email
git config –-global user.name “qianlongwuyongqian”
git config –-global user.email “1201698134@qq.com”
配置成功后，如下图：

（4）生成SSH密钥设置到Github
使用命令ssh-keygen -t rsa -C “1210698134@qq.com”来打开邮箱

设置ssh key到Github ，将id_rsa.pud文件里的代码复制到github->settings->SSH and GPG key->new ssh key来添加key
(5)检验ssh设置是否成功，输入ssh -T git@github.com 命令进行检验

4.安装hexo插件
输入代码
cd /
pwd /
npm install hexo-cli -g
进行安装，安装结束后如下图所示：

之后再输入代码：
hexo init Hexo

cd /Hexo
npm instal

hexo g

hexo s

5.上传hexo博客的静态文档到GitHub
(1)安装部署到GitHub插件依赖
npm install -save hexo-deployer-git

(2)修改Hexo文件夹中_config.yml的配置，要修改的内容以及修改后的结果如下图:

（3）将GitHub部署hexo到github
hexo d
部署成功会出现下图结果：

6.打开qianlongwuyongqian.github.io,如果出现了如下的界面，则代表博客已经成功的搭建好了。
